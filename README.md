# Amigo Secreto App

Hola y bienvenido (a)! Este es **mi primer proyecto web**, desarrollado como parte de un challenge en la clase de 
"Principiante en Programacion" de Alura Latam. Es una excelente forma de practicar y poner en práctica HTML, CSS, JavaScript y subir el proyecto a GitHub.  
El challenge consiste en agregar nombres de personas y luego hacer un sorteo para elegir un "amigo secreto" de manera aleatoria.

---

## ¿Cómo Funciona?

1. Escribe el nombre de un amigo y haz clic en **"Añadir"**.
2. Repite este paso para cada persona que deseas incluir.
3. Cuando tengas la lista, haz clic en **"Sortear amigo"** y la aplicacion mostrará un nombre elegido al azar.
